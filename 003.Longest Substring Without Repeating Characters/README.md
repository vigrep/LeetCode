# 问题描述
从一个字符串中找出不包含重复字符的最长的子串长度。

**举例:**
```
1. "abcabcbb": 不包含重复字符的最长子串为: "abc", length: 3
2. "bbbbbb": 不包含重复字符的最长子串为: "b"，length: 1
3. "pwwkew": 不包含重复字符的最长子串为: "wke", length: 3
```
