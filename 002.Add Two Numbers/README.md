# Add Two Numbers

LeetCode链接：[Add Two Numbers](https://leetcode.com/problems/add-two-numbers/description/)

### 问题描述

用两个非空的链表代表两个非负整数，数字是逆序存储在链表中, 并且每一个节点只能存储一个数字(0-9),返回两个链表相加的结果。

注：模拟加法运算/满十进一

**举例:**

```
Input: (2 -> 4 -> 3) + (3 -> 6 -> 4)
Output: 5 -> 0 -> 8

342 + 463 = 805
```
